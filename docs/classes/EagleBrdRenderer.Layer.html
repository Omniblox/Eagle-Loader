EagleBrdRenderer.Layer Class

Defined in: src/eagle-brd-renderer.js:3436
Module: EagleBrdRenderer

Define a layer of a PCB board. This contains a list of elements, and the render of these elements.

Layer objects do not correspond directly to BRD layers. They may combine many BRD layers into one target, such as for rendering silkscreen; share an element across multiple targets, such as with vias and holes; or ignore a BRD layer entirely, such as with Symbol layers.
